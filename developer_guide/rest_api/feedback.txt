Feedback
--------
The feedback API allows you to create, retrieve, update, and search Feedback objects.

Create
======
**POST /feedback/create.json**

Creates a new feedback entry.

==================      ============================================
Field                   Type
==================      ============================================
``author_address``      string
``created``             timestamp
``quote_mood``          string: ``praise``/``neutral``/``criticism``
``quote_text``          string
``url``                 string
``custom_*``            mixed
==================      ============================================

.. code-block:: php

    <?php
    $postfields = array(
        array('quote_text','This is the best app ever!'),
        array('author_address','happy-user@example.com'),	
    );
    $out = $cerb->post($base_url . 'feedback/create.json', $postfields);

Retrieve
========
**GET /feedback/<id>.json**

Retrieves information about a feedback entry.

.. code-block:: php

    <?php
    $out = $cerb->get($base_url . 'feedback/1.json');
	
Update
======
**PUT /feedback/<id>/xml**

Updates a feedback entry.

==================      ============================================
Field                   Type
==================      ============================================
``author_address``      string
``created``             timestamp
``quote_mood``          string: ``praise``/``neutral``/``criticism``
``quote_text``          string
``url``                 string
``custom_*``            mixed
==================      ============================================

.. code-block:: php

    <?php
    $postfields = array(
        array('url','http://www.google.com/'),
        array('quote_mood','praise'),
    );
    $out = $cerb->put($base_url . 'feedback/1.json', $postfields);
	
Search
======
**POST /feedback/search.json**

Performs a search across feedback objects.

==================      ============================================
Field                   Type
==================      ============================================
``id``                  integer
``author_address``      string
``created``             timestamp
``quote_mood``          string: ``praise``/``neutral``/``criticism``
``quote_text``          string
``url``                 string
==================      ============================================

.. code-block:: php

    <?php
    $postfields = array(
        array('criteria[]','author_address'),
        array('oper[]','like'),
        array('value[]','a*'),
        array('sortBy','id'),
        array('sortAsc','1'),
        array('page','1'),
    );
    $out = $cerb->post($base_url . 'feedback/search.json', $postfields);
