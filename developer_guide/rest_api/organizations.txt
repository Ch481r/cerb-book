Organizations
-------------
The organizations API allows you to create, retrieve, update, and search Organization objects.

Create
======
**POST /orgs/create.json**

Creates a new organization.

============      =========
Field             Type     
============      =========
``name``          string   
``street``        string   
``city``          string   
``province``      string   
``postal``        string   
``country``       string   
``phone``         string   
``website``       string   
``created``       timestamp
``custom_*``      mixed    
============      =========

.. code-block:: php

    <?php
    $postfields = array(
        array('name','deCODE Genetics'),
    );
    $out = $cerb->post($base_url . 'orgs/create.json', $postfields);
	
Retrieve
========
**GET /orgs/<id>.json**

Retrieves information about an organization.

.. code-block:: php

    <?php
    $out = $cerb->get($base_url . 'orgs/1.json');

Search
======
**POST /orgs/search.json**

Performs a search across organization objects.

============    =========
Field           Type     
============    =========
``id``          integer  
``name``        string   
``street``      string   
``city``        string   
``province``    string   
``postal``      string   
``country``     string   
``phone``       string   
``website``     string   
``created``     timestamp
``custom_*``    mixed    
============    =========

.. code-block:: php

    <?php
    $postfields = array(
        array('criteria[]','name'),
        array('oper[]','like'),
        array('value[]','t*'),
        array('sortBy','name'),
        array('sortAsc','1'),
        array('page','1'),
    );
    $out = $cerb->post($base_url . 'orgs/search.json', $postfields);
