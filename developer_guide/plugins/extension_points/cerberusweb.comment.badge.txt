cerberusweb.comment.badge
------------------------

The **cerberusweb.comment.badge** extension point allows you to display an image next to comments left by a worker.

Plugin Manifest
===============

In :doc:`plugin.xml </developer_guide/plugins/plugin.xml>`:

.. code-block:: xml
    
    <!-- Comment Badge -->
    <extension point="cerberusweb.comment.badge">
        <id>example.comment.badge</id>
        <name>Example Comment Badge</name>
        <class>
            <file>api/App.php</file>
            <name>ExCommentBadge</name>
        </class>
        <params/>
    </extension>
    
Implementation
==============
.. code-block:: php

    <?php
    abstract class Extension_CommentBadge
        extends DevblocksExtension {
            
            function render(Model_Comment $comment) {}
    };

**render()**
    Renders the HTML next to the worker comments. The comment being rendered is available in the ``$comment`` variable so you can determine information about the comment. See the documentation on :doc:`Model_Comment </developer_guide/devblocks/models/model_comment>` for more information on the data and methods you have access to for each comment.
