cerberusweb.task.toolbaritem
------------------------

The **cerberusweb.task.toolbaritem** extension point allows you to add content to the task toolbar. This allows plugins to contribute functionality that assists with handling tasks.

Plugin Manifest
===============

In :doc:`plugin.xml </developer_guide/plugins/plugin.xml>`:

.. code-block:: xml
    
    <!-- Ticket Toolbar -->
    <extension point="cerberusweb.task.toolbaritem">
        <id>example.task.toolbaritem</id>
        <name>Example Task Toolbar Item</name>
        <class>
            <file>api/App.php</file>
            <name>ExampleTicketToolbarItem</name>
        </class>
        <params/>
    </extension>

Implementation
==============
.. code-block:: php

    <?php
    abstract class Extension_TicketToolbarItem
        extends DevblocksExtension {
            
            function render(Model_Task $task) { }
    };

**render()**
    Renders the content for the toolbar. The method is passed the task being displayed as the ``$task`` parameter. See the documentation on :doc:`Model_Task </developer_guide/devblocks/models/model_task>` for more information on the data and methods you have access to for each task.
