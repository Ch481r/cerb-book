cerberusweb.support_center.message.badge
------------------------

The **cerberuswebsupport_center.message.badge** extension point allows you to display an image next to messages in the :doc:`Support Center`.

Plugin Manifest
===============

In :doc:`plugin.xml </developer_guide/plugins/plugin.xml>`:

.. code-block:: xml
    
    <!-- Support Center Message Badge -->
    <extension point="cerberusweb.support_center.message.badge">
        <id>example.support_center.message.badge</id>
        <name>Example Support Center Message Badge</name>
        <class>
            <file>api/App.php</file>
            <name>ExMessageBadge</name>
        </class>
        <params>
        </params>
    </extension>

Implementation
==============
.. code-block:: php

    <?php
    abstract class Extension_SupportCenterMessageBadge
        extends DevblocksExtension {
            
            function render(Model_Message $message) {}
    };

**render()**
    Renders the HTML next to the worker messages. The message being rendered is available in the ``$message`` variable so you can determine information about the message. See the documentation on :doc:`Model_Message` for more information on the data you have access to for each message.
