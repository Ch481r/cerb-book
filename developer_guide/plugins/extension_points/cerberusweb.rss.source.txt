cerberusweb.rss.source
----------------------

The **cerberusweb.rss.source** extension point allows you to display an image next to comments left by a worker.

Plugin Manifest
===============

In :doc:`plugin.xml </developer_guide/plugins/plugin.xml>`:

.. code-block:: xml
    
    <!-- RSS Source -->
    <extension point="cerberusweb.rss.source">
        <id>cerberusweb.rss.source.example</id>
        <name>Example</name>
        <class>
            <file>api/App.php</file>
            <name>ExampleRssSource</name>
        </class>
        <params>
        </params>
    </extension>

Implementation
==============
.. code-block:: php

    <?php
    class ExampleRssSource
        extends Extension_RssSource {
            
            function getFeedAsRss($feed) {}
    };

    [edit]

**getFeedAsRss()**
    This method is called when a feed is loaded. This should take the ``$feed`` parameter and return XML in the RSS format. See the documentation on :doc:`Model_ViewRSS </developer/devblocks/models/model_viewrss>` for more information on the data and methods you have access to for each feed.
