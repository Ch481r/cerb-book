cerberusweb.ticket.toolbaritem
------------------------------

The **cerberusweb.ticket.toolbaritem** extension point allows you to add content to the ticket toolbar. This allows plugins to contribute functionality that assists with responding to tickets.

Plugin Manifest
===============

In :doc:`plugin.xml </developer_guide/plugins/plugin.xml>`:

.. code-block:: xml
    
    <!-- Ticket Toolbar -->
    <extension point="cerberusweb.ticket.toolbaritem">
        <id>example.ticket.toolbaritem</id>
        <name>Example Ticket Toolbar Item</name>
        <class>
            <file>api/App.php</file>
            <name>ExampleTicketToolbarItem</name>
        </class>
        <params/>
    </extension>

Implementation
==============
.. code-block:: php

    <?php
    class ExampleTicketToolbarItem
        extends Extension_TicketToolbarItem {
            
            function render(CerberusTicket $ticket) { }
    };

**render()**
    Renders the content for the toolbar. The method is passed the ticket being displayed as the ``$ticket`` parameter. See the documentation on :php:class:`CerberusTicket` for more information on the data and methods you have access to for each task.
