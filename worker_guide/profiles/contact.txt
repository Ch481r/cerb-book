Contact
-------

The profile page is divided into two sections. The top section of the profile page provides you quick access to the properties of the organization and a toolbar which allows you to quickly perform actions on the ticket.  The bottom section of the page is a group of tabs which control what content is displayed.

.. figure:: /images/worker_guide/profiles/contact/profile.png
    :alt: The Registered Contact profile page.
    :figclass: bordered

Properties
==========

The Properties section displays the **email**, **created**, and **last login** of a Registered Contact. Any custom fields that are set for a Registered Contact are also displayed in this section.

.. figure:: /images/worker_guide/profiles/contact/properties.png
    :alt: The properties section of a Registered Contact profile page.
    :figclass: bordered

Actions
=======

The actions toolbar has three buttons.

.. figure:: /images/worker_guide/profiles/contact/actions.png
    :alt: The toolbar on a Registered Contact profile page.
    :figclass: bordered

Start Watching
~~~~~~~~~~~~~~

The :guilabel:`Start watching` button allows you to watch a Registered Contact and see how many workers are currently watching it.

.. figure:: /images/worker_guide/profiles/contact/actions/start_watching.png
    :alt: The start watching button in the toolbar.
    :figclass: bordered

The :guilabel:`â–¾` button expands the watchers peek so you can see which workers are watching a Registered Contact. The watchers peek also allows you to add and remove watchers from a Registered Contact.

.. figure:: /images/worker_guide/profiles/contact/actions/watchers_dialog.png
    :alt: The watchers peek.
    :figclass: bordered

Edit
~~~~

The :guilabel:`Edit` button allows you to edit a Registered Contact.

.. figure:: /images/worker_guide/profiles/contact/actions/edit.png
    :alt: The edit button in the toolbar.
    :figclass: bordered

Clicking the :guilabel:`Edit` button displays the edit dialog.

.. figure:: /images/worker_guide/profiles/contact/actions/edit_dialog.png
    :alt: The edit dialog.
    :figclass: bordered

The **password**, and **watchers** can be specified in the dialog.  If any custom fields have been defined for registered contacts, they can be specified as well.

Time Tracking
~~~~~~~~~~~~~

If you have the :doc:`Time Tracking</worker_guide/search/time_tracking>` plugin enabled, the :guilabel:`Track Time` button will allow you to track time against a Registered Contact.

.. figure:: /images/worker_guide/profiles/contact/actions/track_time.png
    :alt: The track time button in the toolbar.
    :figclass: bordered

Clicking the :guilabel:`Track Time` button will open the Time Tracking overlay - the :doc:`documentation on Time Tracking </worker_guide/search/time_tracking>` has more information.

Tabs
====

The tabs on a Registered Contact profile page in the bottom section can be used to view the Activity Log, Comments, Links, Mail History, and People. Some plugins also add additional tabs to the profile page, such as :doc:`Opportunities </worker_guide/search/opportunities>`.

.. figure:: /images/worker_guide/profiles/contact/tabs.png
    :alt: The tabs on a Registered Contact profile page.
    :figclass: bordered

Activity Log
~~~~~~~~~~~~

The :guilabel:`Activity Log` tab displays a record of all the changes made to a Registered Contact.

.. figure:: /images/worker_guide/profiles/contact/tabs/activity_log.png
    :alt: The Activity Log tab on a Registered Contact profile page.
    :figclass: bordered

Comments
~~~~~~~~

The :guilabel:`Comments` tab shows you all of the comments set on a Registered Contact, and allows you to write a new comment.

.. figure:: /images/worker_guide/profiles/contact/tabs/comments.png
    :alt: The Comments tab on a Registered Contact profile page.
    :figclass: bordered

Links
~~~~~

The :guilabel:`Links` tab displays all of the objects currently linked to a Registered Contact. The drop-down fields at the top of the tab can be used to quickly link existing objects to a Registered Contact, and create and link new ones.

.. figure:: /images/worker_guide/profiles/contact/tabs/links.png
    :alt: The Links tab on a Registered Contact profile page.
    :figclass: bordered

Email Addresses
~~~~~~~~~~~~~~~

The :guilabel:`Email Addresses` tab displays all of the email addresses associated with a Registered Contact.

.. figure:: /images/worker_guide/profiles/contact/tabs/email_addresses.png
    :alt: The Email Addresses tab on a Registered Contact profile page.
    :figclass: bordered

Mail History
~~~~~~~~~~~~

The :guilabel:`Mail History` tab displays the ticket history for a Registered Contact.

.. figure:: /images/worker_guide/profiles/contact/tabs/mail_history.png
    :alt: The Mail History tab on a Registered Contact profile page.
    :figclass: bordered
