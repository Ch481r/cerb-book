Organizations
-------------

The **organizations** tab is where you add and view organizations.

.. figure:: /images/worker_guide/address_book/tabs/organizations.png
    :alt: The organizations tab.
    :figclass: bordered

Toolbar
=======
The toolbar at the top of the organizations tab contains two buttons and the quick search box.

.. figure:: /images/worker_guide/address_book/tabs/organizations_toolbar.png
  :alt: The orgs toolbar.
  :figclass: bordered

Add Org
~~~~~~~~

The :guilabel:`Add Org` button at the top of the organizations tab allows you to add organizations.

.. figure:: /images/worker_guide/address_book/tabs/organizations_add.png
    :alt: The add org button.
    :figclass: bordered

Clicking the :guilabel:`Add Task` button will open the add organization dialog.

.. figure:: /images/worker_guide/address_book/tabs/organizations_dialog_add.png
    :alt: The add organization dialog.
    :figclass: bordered

.. ::
     Link to comments doc, update image for customf ields

The **name**, **street**, **city**, **state/province**, **postal**, **country**, **phone**, **website**, and **watchers** can all be specified in the dialog.  If any custom fields have been defined for organizations, they can be specified as well. There is also a comment box so you can quickly leave a comment for the organization when you create it.

Merge
~~~~~

The :guilabel:`Merge` button at the top of the organizations tab allows you to merge organizations.

.. figure:: /images/worker_guide/address_book/tabs/organizations_merge.png
    :alt: The merge button.
    :figclass: bordered

Clicking the :guilabel:`Merge` button will open the merge organization dialog.

.. figure:: /images/worker_guide/address_book/tabs/organizations_dialog_merge.png
    :alt: The merge dialog.
    :figclass: bordered

The organization entry field will auto-complete as you type, and clicking the magnifying glass icon will open the organizations :doc:`chooser </concepts/ui/choosers>`. Clicking the :guilabel:`continue` button will display another dialog which allows you to pick which values to use for each field.

.. figure:: /images/worker_guide/address_book/tabs/organizations_dialog_merge_select.png
    :alt: The merge dialog.
    :figclass: bordered

Clicking the :guilabel:`continue` button will merge the selected organizations into a single organization.

Quick Search
~~~~~~~~~~~~

The :doc:`quick search </concepts/objects/quick_search>` box is a time-saving feature that allows you to quickly apply a filter to the worklist.

.. figure:: /images/worker_guide/address_book/tabs/organizations_quicksearch.png
    :alt: The quick search box.
    :figclass: bordered

Worklist
========

The :doc:`worklist </concepts/worklists/index>` is displayed just below the tabs.  At the top of the worklist are the currently applied :doc:`filters </concepts/worklists/filters>`.  Just below the filters, on the right side of the worklist, you have access to :doc:`subtotals </concepts/worklists/subtotals>`, :doc:`copy </concepts/worklists/copy>`, and :doc:`export </concepts/worklists/export>`.  Beneath the worklist is a toolbar of :doc:`actions </concepts/worklists/actions>` you can perform on the selected results.

.. figure:: /images/worker_guide/address_book/tabs/organizations.png
    :alt: The tasks worklist with subtotals active.
    :figclass: bordered
