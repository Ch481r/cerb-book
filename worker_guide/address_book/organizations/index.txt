Display
-------

The display page is divided into two sections. The top section of the display page provides you quick access to the properties of the organization and a toolbar which allows you to quickly perform actions on the ticket.  The bottom section of the page is a group of tabs which control what content is displayed.

.. figure:: /images/worker_guide/address_book/organizations/display.png
    :alt: The organizations display page.
    :figclass: bordered

Properties
==========

The properties section displays the **name**, **street**, **city**, **state/province**, **postal**, **country**, **phone**, **website**, and **created** of the organization. Any custom fields that are set for the organization are also displayed in this section.

.. figure:: /images/worker_guide/address_book/organizations/properties.png
    :alt: The properties section of the display page.
    :figclass: bordered

Actions
=======

The actions toolbar has two buttons.

.. figure:: /images/worker_guide/address_book/organizations/actions.png
    :alt: The toolbar section of the display page.
    :figclass: bordered

Start Watching
~~~~~~~~~~~~~~

The :guilabel:`Start watching` button allows you to watch the organization and see how many workers are currently watching it.

.. figure:: /images/worker_guide/address_book/organizations/actions/startwatching.png
    :alt: The start watching button.
    :figclass: bordered

The :guilabel:`â–¾` button expands the watchers peek so you can see which workers are watching the organization. The watchers peek also allows you to add and remove watchers from the organization.

.. figure:: /images/worker_guide/address_book/organizations/actions/watchers_dialog.png
    :alt: The watchers peek.
    :figclass: bordered
    
Edit
~~~~

The :guilabel:`Edit` button allows you to edit the organization.

.. figure:: /images/worker_guide/address_book/organizations/actions/edit.png
    :alt: The edit button.
    :figclass: bordered


Clicking the :guilabel:`Edit` button displays the edit dialog. The edit dialog has 3 tabs.

Properties
""""""""""

The :guilabel:`Properties` tab can be used to edit the properties and custom fields on the organization, and quickly leave a comment.

.. figure:: /images/worker_guide/address_book/organizations/actions/edit_dialog_properties.png
    :alt: The properties tab in the edit dialog.
    :figclass: bordered

People
""""""

The :guilabel:`People` tab can be used to view and add contacts to the organization.

.. figure:: /images/worker_guide/address_book/organizations/actions/edit_dialog_people.png
    :alt: The people tab in the edit dialog.
    :figclass: bordered

Comments
""""""""

The comments tab shows you all of the comments set on the organization, and allows you to write a new comment.

.. figure:: /images/worker_guide/address_book/organizations/actions/edit_dialog_comments.png
    :alt: The comments tab in the edit dialog.
    :figclass: bordered

Tabs
====

The tabs in the bottom section can be used to view the Activity Log, Comments, Links, Mail History, and People. Some plugins also add additional tabs to the display page, such as :doc:`Opportunities </worker_guide/activity/tabs/opportunities>`.

.. figure:: /images/worker_guide/address_book/organizations/tabs.png
    :alt: The tabs.
    :figclass: bordered

Activity Log
~~~~~~~~~~~~

The Activity Log tab displays a record of all the changes made to an organization.

.. figure:: /images/worker_guide/address_book/organizations/tabs/activity_log.png
    :alt: The activity log tab.
    :figclass: bordered

Comments
~~~~~~~~

The comments tab shows you all of the comments set on the organization, and allows you to write a new comment.

.. figure:: /images/worker_guide/address_book/organizations/tabs/comments.png
    :alt: The comments tab.
    :figclass: bordered


Links
~~~~~

The Links tab displays all of the objects currently linked to the organization you are viewing. The drop-down fields at the top of the tab can be used to quickly link existing objects to the organization, and create and link new ones.

.. figure:: /images/worker_guide/address_book/organizations/tabs/links.png
    :alt: The links tab.
    :figclass: bordered

Mail History
~~~~~~~~~~~~

The Mail History tab displays the ticket history for the organization.

.. figure:: /images/worker_guide/address_book/organizations/tabs/mail_history.png
    :alt: The mail history tab.
    :figclass: bordered

The :guilabel:`Send Mail` button at the top of the Mail History tab allows you to easily send a new message.

.. figure:: /images/worker_guide/address_book/organizations/tabs/send_mail.png
    :alt: The Send Mail button on the Mail History tab.
    :figclass: bordered

Clicking the :guilabel:`Send Mail` button opens the :guilabel:`Compose` dialog, allowing you to compose a new message.

.. figure:: /images/worker_guide/address_book/organizations/tabs/compose.png
    :alt: The Compose dialog.
    :figclass: bordered

People
~~~~~~

The People tab displays all of the :doc:`contacts</worker_guide/address_book/tabs/addresses>` that are assigned to an organization, and allows you to quickly add a new contact.

.. figure:: /images/worker_guide/address_book/organizations/tabs/people.png
    :alt: The Add Contact button on the People tab.
    :figclass: bordered

The :guilabel:`Add Contact` button allows you to add a new contact to the organization.

.. figure:: /images/worker_guide/address_book/organizations/tabs/add_contact_button.png
    :alt: The Add Contact button on the People tab.
    :figclass: bordered

Clicking the :guilabel:`Add Contact` button opens the :guilabel:`Email Contact` dialog.

.. figure:: /images/worker_guide/address_book/organizations/tabs/email_contact.png
    :alt: The Email Contact dialog.
    :figclass: bordered
