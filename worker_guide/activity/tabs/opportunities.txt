Opportunities
-------------

The **opportunities** tab is where you add and view opportunities.

.. figure:: /images/worker_guide/activity/opportunities_tab.png
   :alt: The opportunities tab.
   :figclass: bordered

Toolbar
=======
The toolbar for the opportunities tab contains two buttons and the quick search box.

.. figure:: /images/worker_guide/activity/opportunities_toolbar.png
  :alt: The opportunities toolbar.
  :figclass: bordered

Add Opportunity
~~~~~~~~~~~~~~~

The :guilabel:`Add Opportunity` button at the top of the opportunities tab allows you to add opportunities.

.. figure:: /images/worker_guide/activity/opportunities_toolbar_add.png
   :alt: The add opportunity button.
   :figclass: bordered
   
Clicking the :guilabel:`Add Opportunity` button will open the **add opportunity** dialog.

.. figure:: /images/worker_guide/activity/opportunities_dialog_add.png
   :alt: The add opportunity dialog.
   :figclass: bordered

.. ::
    Link to comments doc.

The **email**, **title**, **status**, **amount**, **created date**, and **watchers** can all be specified in the dialog.  If any custom fields have been defined for opportunities, they can be specified as well. There is also a comment box so you can quickly leave a comment for the opportunity when you create it.

Import
~~~~~~

The :guilabel:`Import` button allows you to import a CSV file.

.. figure:: /images/worker_guide/activity/opportunities_toolbar_import.png
   :alt: The import button.
   :figclass: bordered
   
Clicking the :guilabel:`Import` button will open the **import** dialog.

.. figure:: /images/worker_guide/activity/opportunities_dialog_import.png
   :alt: The import dialog.
   :figclass: bordered

The dialog allows you to specify a CSV file to upload.

.. figure:: /images/worker_guide/activity/opportunities_import.png
   :alt: The add opportunity button.
   :figclass: bordered
   
.. ::
    Link to custom field docs below.
    
The columns present in the CSV file can be imported as data for any of the fields available for opportunities, including custom fields.

Quick Search
~~~~~~~~~~~~

The quick search box allows you to quickly apply a filter to the worklist.

.. figure:: /images/worker_guide/activity/opportunities_toolbar_search.png
   :alt: The quick search box.
   :figclass: bordered

Worklist
========

The :doc:`worklist </concepts/worklists/index>` is displayed just below the tabs.  At the top of the worklist are the currently applied :doc:`filters </concepts/worklists/filters>`.  Just below the filters, on the right side of the worklist, you have access to :doc:`subtotals </concepts/worklists/subtotals>`, :doc:`copy </concepts/worklists/copy>`, and :doc:`export </concepts/worklists/export>`.  Beneath the worklist is a toolbar of :doc:`actions </concepts/worklists/actions>` you can perform on the selected results.

.. figure:: /images/worker_guide/activity/opportunities.png
   :alt: The opportunities worklist with subtotals active.
   :figclass: bordered
