Server Requirements
-------------------

If you are unable to meet these requirements, consider the On-Demand edition, or utilizing a cloud-based virtual private server from a vendor like `Linode <http://www.linode.com/>`_.

Required
========

- Any of these operating systems:
    - :program:`Linux`
    - :program:`BSD`
    - :program:`Mac OS X`
    - :program:`Windows`
- Any of these webserver applications:
    - :program:`Apache` 2.x
    - :program:`Nginx`
    - :program:`lighttpd`
    - :program:`IIS`
- :program:`PHP` 5.3 or later
    - With the following extensions enabled:
        - curl
        - dom
        - gd
        - imap
        - json
        - mailparse
        - mbstring
        - mysql
        - pcre
        - session
        - simplexml
        - spl
        - xml
    - With the :file:`php.ini` settings:
        - `file_uploads = On`
- :program:`MySQL` 5.0 or later

Recommended
===========

- :program:`Linux` 
    - :program:`Ubuntu` (10.04 LTS or 11.10) or :program:`Debian` (6.0 Squeeze)
    - :program:`CentOS` is supported but not recommended due to outdated packages
- :program:`Apache` 2.x or later
- :program:`PHP` 5.3 or later
    - With the required extensions, plus:
        - memcached
        - xcache
    - With the :file:`php.ini` settings:
        - `file_uploads = On`
        - `magic_quotes_gpc = Off`
        - `safe_mode = Off`
        - `memory_limit = 32M` (or higher)
- :program:`MySQL` 5.x, with:
    - `Strict mode <http://dev.mysql.com/doc/refman/5.0/en/server-sql-mode.html#sqlmode_strict_all_tables>`_ disabled
- :program:`Memcached`
